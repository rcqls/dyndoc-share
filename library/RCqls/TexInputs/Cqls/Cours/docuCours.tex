% Ceci est rendu possible en suivant les instructions suivantes :
% - placer la commande  ``\input inputCours.tex'' en préambule du document tex
% - juste après (resp. avant) \begin{document} (resp. \end{document}) inclure 
% la commande ``\begin{Cours}'' (resp. ``\end{Cours}''.
%
% - Description des commandes suivantes :
% - \QueTransparent{} (resp. \QuePapier{}) : sorties exclusives transparent pdf (resp. document dvi)
% - \NewPage remplace le rôle de \newpage dans le but de ne générer le saut de page que dans le cas 
% de document transparent (pdf)
% - \Cible{nomCible}, \Lien{nomCible}{texteInteractif} sont utilisés pour faire des hyperliens (nomCible et 
% texteInteractif) étant bien entendu à renommer pour chaque hyperlien.
% - \Boite{\ldots} ,\BoiteCentre{\ldots} (i.e. \centerline{\Boite{\ldots}): permettent de mettre en valeur une ligne 
% ou une partie de ligne.
% - \BoiteGrosse{\ldots} : correspond à BoiteCentre{\minipage{10cm}{\ldots}\end{minipage})
% - RougeClair, VertClair, Vert, BleuClair, RougeFonce, VertFonce, BleuFonce sont des couleurs utilisables 
% pour la définition des commandes ci-dessous.
% - \BoiteCouleur{couleur}{\ldots}, \BoiteCentreCouleur{couleur}{\ldots}, \BoiteGrosseCouleur{couleur}{\ldots} : extensions 
% colorisées de  \Boite{\ldots} ,\BoiteCentre{\ldots} et \BoiteGrosse{\ldots}
% - \BoiteCouleurTaille{couleur}{taille}{\ldots} : équivalent de \BoiteGrosseCouleur{couleur}{\ldots} avec possibilité 
% de changement de la taille 10cm par ``taille''.
%
